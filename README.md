# 班级抽签/抽号数用小程序
可以用于班级抽签/抽号数<br/>
可以设置自己想要的范围，点击开始即开始随机生成一个范围内的号数。<br/>
默认范围是1-50，不能设置为负数，并且数值不能超过4294967296即2^32。<br/>
